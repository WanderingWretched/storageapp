Инструкция для запуска приложения
 Способ № 1
Скачать репозиторий. Создать локальную базу данных с названием: storageapp, и настроить файл properties для запуска

 Способ № 2
1) Скачать докер репозиторий командой: docker pull ghcr.io/wanderingwretched/storageapp:v1.0.0
2) В терминале прописать команду docker-compose up -d
3) Запустить приложение командой docker run ghcr.io/wanderingwretched/storageapp:v1.0.0
